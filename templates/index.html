About this project

A Flask + TextBlob web app built as a Python Development internship project. It classifies text as Positive, Negative, or Neutral, and reports polarity and subjectivity scores from TextBlob's NLP model.

Frontend: Bootstrap 5. Backend: Flask, TextBlob.
